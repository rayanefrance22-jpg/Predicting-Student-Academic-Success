# Predicting Student Academic Success Using Statistical Modeling

## Overview

This project aims to predict student academic success using statistical modeling and machine learning techniques. The model analyzes demographic, academic, and behavioral data from Excel datasets to estimate the probability of student success or academic risk.

## Objective

The objective is to develop a reliable prediction model that helps educational institutions identify students who may require academic support at an early stage. The project demonstrates the complete data science workflow, from data preprocessing to model evaluation.

## Features

- Read student data from Excel files
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Machine learning model training
- Prediction of academic outcomes
- Performance evaluation
- Data visualization

## Dataset

The input dataset contains student demographic, academic, and behavioral information stored in Excel format. Each row represents one student, while each column represents a feature used for prediction.

## Expected Output

The model predicts the probability of academic success or failure for each student and provides evaluation metrics to measure prediction performance.
